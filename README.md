# calorie_counter
🥗 Calorie Counter App

A simple and interactive Calorie Counter Web App built with JavaScript. This app allows users to track their daily food intake, manage calorie consumption, and store data locally in the browser.

🚀 Features
✅ Add food items with calorie values
📋 Display a list of all added foods
❌ Remove individual food items
🔢 Automatically calculate total calories
🔄 Reset daily calorie count
💾 Persist data using localStorage (data remains after refresh)
🛠️ Technologies Used
HTML5
CSS3 / Tailwind / Bootstrap (optional styling)
JavaScript (Vanilla JS)
Browser localStorage API

⚙️ How It Works
User enters a food item and its calorie value
The app stores the item in an array
The list is displayed dynamically on the page
Total calories are calculated automatically
Data is saved in localStorage so it persists
